# プロエン制作物まとめ

## お題：弾幕シューティング  
自機を操作し、相手の弾幕攻撃をよけながら敵を倒すゲーム

---

## 基本実装項目

- **残機・ボムの概念追加**  
  - 残機制によるゲームオーバー判定  
  - ボム効果は前回制作したものを流用  

- **スコア仕様の拡張**  
  - 攻撃命中時にスコア獲得  
  - 敵撃破ボーナスの追加  

- **攻撃力・HPの追加**  
  - 自機および敵にHPを設定  
  - 攻撃力に応じたダメージ計算  

- **自動連射機能の追加**  
  - ショットキー長押しで連射  
  - 連射間隔の制御  

- **ボスキャラの作成**  
  - 一定条件で出現  
  - 通常敵より高HP・特殊攻撃を持つ  

- **BGMの追加**  
  - ゲーム中にBGM再生  
  - ボス戦専用BGMの切り替え  

- **UI作成**  
  - 残機数、ボム数、スコア、HPゲージの表示
  - 背景、弾の変更  

---

## 高難度実装項目

- **キャラ性能差の実装**  
  - ショット・ボム性能が異なるキャラ  
  - キャラクターピック画面の作成  

- **ステージ概念の追加**  
  - 敵配置パターンの設定  
  - ゴール条件の設定 
 

- **ボスHP比例効果（スペルカード）**  
  - HPに応じて攻撃パターン変化  
  - 必殺技（スペルカード）の実装  

- **アイテムドロップ**  
  - 敵撃破時にアイテムドロップ  
  - アイテム効果の設定  
  - 上部回収システムの実装  

- **パワー概念の追加**  
  - アイテム回収によるショット強化  
  - パワー段階による攻撃変化  

- **低速モード・高速モード**  
  - 移動速度の切り替え  
  - モードごとのショット性能変化
